Sharedoc is a realtime document editing application

1. To run this, first extract sharedoc.zip into its own folder.
2. Navidate to sharedoc/ src/ component/ peers/
3. On One machine, open send.html. On other machine, open receive.html.
4. Once two pages are loaded. Copy ID from Receive machine and click on connect button.
5. Two browsers will report connected to ID xxxxxxxxxxxxxxx.
6. Start working on the document and see two document update in real time.
